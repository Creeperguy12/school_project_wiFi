wiFi School project

By Matthew Adlam

What is wiFi?

wiFi is a facility allowing devices to connect to the internet or communicate with others wirelessly in another area. It is WLAN (wireless local area network). It is essentially a wireless Ethernet.

Why is wiFi important

We need wiFi because otherwise we would require a lot of Ethernet ports everywhere, and for a school, event or workplace this would be impractical, and this would also be expenisve. The solution is to use wiFi.
